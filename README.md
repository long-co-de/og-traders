# ODG Traders

A premium mobile-first frontend for **ODG Traders**, a forex trading mentorship platform focused on trading signals, education, subscriptions, and user account management.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- React Router DOM
- Lucide React

## Features

- Authentication (Landing, Login & Sign Up)
- Home Dashboard
- Trading Signals
- Learning & Course Details
- Subscription Plans
- Notifications
- Profile & Settings
- Responsive Mobile & Desktop Layout

## Project Structure

```text
src/
├── assets/
├── components/
├── pages/
├── App.tsx
├── index.css
└── main.tsx
```

## Design Approach

The goal was to create a clean, premium, and intuitive trading experience using a mobile-first approach while maintaining a consistent design system and responsive layout.

## Getting Started

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```
